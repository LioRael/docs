## 快速开始

使用命令 `/ze open` 或 `/zephyrion open` 打开工作空间界面，开始使用 Zephyrion。

## 核心概念

### 工作空间

工作空间是物品存储的顶级容器，类似于一个"文件夹"，可以包含多个保险库。

### 保险库

保险库是实际存储物品的地方，类似于一个超大的箱子，支持分页浏览。

### 配额

配额限制了你可以创建的工作空间数量和使用的存储槽位总数。

## 界面通用操作

| 操作         | 说明         |
| ---------- | ---------- |
| 左键点击       | 进入/选择/确认   |
| 右键点击       | 管理/编辑/删除   |
| Shift + 点击 | 批量操作（部分界面） |
