## 工作空间类型

### 公共工作空间 (PUBLIC)

- **特点**：可邀请其他玩家加入
- **物品共享**：所有成员共享保险库内的物品
- **适用场景**：团队协作、公会仓库、共享资源

**权限要求**：`zephyrion.create-public-workspace`

### 私有工作空间 (PRIVATE)

- **特点**：仅创建者可访问
- **物品私有**：只有你能看到和使用其中的物品
- **适用场景**：个人物品存储、私人收藏

**权限要求**：`zephyrion.create-private-workspace`

### 独立工作空间 (INDEPENDENT)

- **特点**：所有玩家都能看到，但物品相互独立
- **物品独立**：每个玩家在同一保险库中有独立的存储空间
- **适用场景**：服务器公共仓库系统

**权限要求**：`zephyrion.create-independent-workspace`

> 独立工作空间需要服务器管理员在配置中启用 `workspace.independent: true`

## 创建工作空间

### 步骤

1. 打开工作空间列表 (`/ze open`)
2. 点击底部的「创建工作空间」按钮
3. 进入创建界面

### 创建界面

```
┌─────────────────────────────────────────┐
│  创建工作空间                            │
├─────────────────────────────────────────┤
│                                         │
│     [信息预览]                           │
│                                         │
│  [设置名称]  [设置描述]  [选择类型]        │
│                                         │
├─────────────────────────────────────────┤
│          [返回]    [确认]                │
└─────────────────────────────────────────┘
```

### 设置工作空间名称

点击「设置名称」后，在聊天框输入名称。

**名称规则：**

- 长度：3-16 个字符
- 不能包含空格
- 不能使用颜色代码（除非服务器允许）
- 不能使用黑名单中的词汇
- 同一玩家下不能重名

### 设置工作空间描述

点击「设置描述」后，在聊天框输入描述。描述是可选的，用于说明工作空间的用途。

### 选择工作空间类型

点击「选择类型」后，会弹出类型选择界面：

```
┌─────────────────────────────────────────┐
│  选择工作空间类型                         │
├─────────────────────────────────────────┤
│                                         │
│  [公共]      [私有]      [独立]          │
│                                         │
├─────────────────────────────────────────┤
│              [返回]                      │
└─────────────────────────────────────────┘
```

选择你需要的类型，没有对应权限的类型会显示为不可选。

### 确认创建

设置完成后点击「确认」按钮，工作空间创建成功。

## 管理工作空间

### 进入管理界面

在工作空间列表中，**右键点击**工作空间图标进入管理界面。

> 只有工作空间创建者或管理员才能进入管理界面

### 管理界面

```
┌─────────────────────────────────────────┐
│  工作空间管理                            │
├─────────────────────────────────────────┤
│                                         │
│     [工作空间信息]                        │
│                                         │
│  [修改名称]  [修改描述]  [成员列表]        │
│                                         │
│            [删除工作空间]                 │
│                                         │
├─────────────────────────────────────────┤
│              [返回]                      │
└─────────────────────────────────────────┘
```

### 修改名称

1. 点击「修改名称」
2. 在聊天框输入新名称
3. 自动验证并保存

### 修改描述

1. 点击「修改描述」
2. 在聊天框输入新描述
3. 自动保存

### 删除工作空间

1. 点击「删除工作空间」
2. 在聊天框输入 `Y` 确认
3. 输入其他内容或等待超时取消

> **警告**：删除工作空间会同时删除其中所有保险库和物品，此操作不可恢复！

## 成员管理

成员管理仅适用于**公共工作空间**。

### 查看成员列表

在工作空间管理界面点击「成员列表」。

```
┌─────────────────────────────────────────┐
│  成员列表                                │
├─────────────────────────────────────────┤
│  [玩家1头像]  [玩家2头像]  [玩家3头像]     │
│    在线        离线         在线         │
│                                         │
├─────────────────────────────────────────┤
│  [返回] [上一页] [添加] [搜索] [下一页]    │
└─────────────────────────────────────────┘
```

### 添加成员

1. 点击「添加成员」按钮
2. 在聊天框输入要添加的玩家名称
3. 玩家必须在线才能添加

**添加成功后：**

- 新成员会在工作空间列表中看到这个工作空间
- 新成员可以访问工作空间内的所有保险库

### 移除成员

1. 在成员列表中点击要移除的成员头像
2. 确认移除

**限制：**

- 不能移除工作空间创建者
- 只有创建者或管理员可以移除成员

### 搜索成员

支持按玩家名称搜索成员。

## 查看工作空间信息

在工作空间列表中，将鼠标悬停在工作空间图标上可以查看详细信息：

- **ID**：工作空间唯一标识
- **名称**：工作空间名称
- **描述**：工作空间描述
- **类型**：PUBLIC/PRIVATE/INDEPENDENT
- **创建者**：工作空间创建者名称
- **成员**：成员列表
- **创建时间**：创建日期时间
- **更新时间**：最后更新时间

## 注意事项

### 配额限制

创建工作空间会消耗工作空间配额。使用 `/ze quota` 查看当前配额使用情况。

### 类型选择建议

| 场景      | 推荐类型             |
| ------- | ---------------- |
| 个人物品存储  | 私有 (PRIVATE)     |
| 公会/团队仓库 | 公共 (PUBLIC)      |
| 服务器商店系统 | 独立 (INDEPENDENT) |
| 多人共享资源  | 公共 (PUBLIC)      |

### 安全提示

- 公共工作空间的物品对所有成员可见
- 谨慎添加不信任的玩家为成员
- 重要物品建议存放在私有工作空间
