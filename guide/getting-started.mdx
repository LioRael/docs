## 打开主界面

使用以下任一命令打开工作空间列表：

```
/zephyrion open
/ze open
```

## 界面导航

│  [        ] [        ] [        ]       │

## 基本操作流程

### 1. 创建工作空间

1. 打开主界面 (`/ze open`)
2. 点击「创建工作空间」
3. 点击设置名称，输入名称（3-16字符）
4. 点击设置描述（可选）
5. 点击选择类型
6. 点击确认创建

### 2. 创建保险库

1. 进入工作空间
2. 点击「创建仓库」
3. 设置名称和描述
4. 确认创建

### 3. 存放物品

1. 进入保险库
2. 如有锁定槽位，先解锁（点击锁定槽位输入数量）
3. 将物品从背包拖入保险库界面
4. 物品自动保存

### 4. 取出物品

1. 进入保险库
2. 点击要取出的物品
3. 物品自动进入背包

## 搜索功能

### 搜索工作空间

支持按以下条件搜索：

- **名称** - 工作空间名称包含关键词
- **成员** - 成员名称包含关键词
- **描述** - 描述包含关键词

### 搜索物品

在保险库中支持按以下条件搜索：

- **物品名称** - 物品名称包含关键词
- **物品Lore** - Lore内容包含关键词

搜索结果会高亮显示匹配的物品。

## 快捷键说明

| 按键         | 功能          |
| ---------- | ----------- |
| 左键         | 进入/选择/取出物品  |
| 右键         | 管理/放入物品     |
| Shift + 左键 | 快速移动整组物品    |
| Q          | 在某些界面用于取消操作 |
| ESC        | 关闭当前界面      |
