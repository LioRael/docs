## 快速开始

使用命令 `/ze open` 或 `/zephyrion open` 打开工作空间界面。

## 核心概念速览

### 工作空间类型

| 类型 | 说明 |

|------|------|

| 公共 (PUBLIC) | 可邀请成员，物品共享 |

| 私有 (PRIVATE) | 仅创建者可访问 |

| 独立 (INDEPENDENT) | 每人独立存储空间 |

### 基本操作

| 操作 | 说明 |

|------|------|

| 左键点击 | 进入/选择/取出 |

| 右键点击 | 管理/放入/删除 |

| Shift + 点击 | 批量操作 |

### 配额系统

- **工作空间配额**：可创建的工作空间数量
- **存储配额**：可使用的总槽位数
- **无限容量**：不受存储配额限制

查看配额`/ze quota`
